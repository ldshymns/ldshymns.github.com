LDS Hymns for Android

Kindle Users (and others without Android Market): Download the .apk file here

LDS Hymns for Android is in it's first release. Not real fancy, but it works well.

You can look up hymns by number, or search by title and first line.

Music notation is viewable in any pdf viewer you have on the android device. I recommend the adobe viewer as the best free option.

Suggestions, criticism, patches, donations welcome.

The pdf file used for these hymns can be downloaded from lds.org. (link verified 23 Jan 2012)

Spanish version now available on Android Market!
